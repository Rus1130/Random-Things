# - number

put # - puts # to the top of the stack

I/O
in - input as number to top of stack
cin - input as character to top of stack

out - output as number from top of stack
cout - output as character from top of stack

Stack Manipulation
dup - duplicate top of stack
swap - swap top two elements
drop - drop top of stack
