> - shift pointer right
< - shift pointer left

+ - incrememnt cell value
- - decrement cell value

. - output cell value

[ loop ] - if cell value at pointer is not 0, loop
    common loop +++[>+++++<-]

:cell - goto cell if cellValue is 0
;cell - goto cell if cellValue is not 0

use typesets to change the character set
    tD - Decimal
    tB - Binary
    tO - Octal
    tX - Hexadecimal
    tU - Unicode
