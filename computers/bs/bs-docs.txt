> - shift pointer right
< - shift pointer left

+ - incrememnt cell value
- - decrement cell value

, - set cell value
. - output cell value

@ - copy cell value
_ - paste cell value

& - wait for 100ms

[ loop ] - if cell value at pointer is not 0, loop
    common loop +++[>+++++<-]

{cell - goto cell if cellValue is 0
}cell - goto cell if cellValue is not 0
Gcell - goto cell

(value'ifTrue'ifFalse) - if cellValue = value goto ifTrue, else goto ifFalse

use typesets to change the character set
    tD - Decimal
    tB - Binary
    tO - Octal
    tX - Hexadecimal
