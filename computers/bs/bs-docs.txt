> - shift pointer right
< - shift pointer left

+ - incrememnt cell value
- - decrement cell value

. - output cell value

    common loop +++[>+++++<-]

)value - shift pointer right if cellValue equals value
(value - shift pointer left if cellValue equals value

    tB - Binary
